    Kebutuhan Sistem¶

    Alur kerja aplikasi¶

    Aplikasi monitoring suhu dan kelembaban ruangan greenhouse memiliki alur sistem seperti berikut:

    • Letakkan mikrokontroller arduino uno yang sudah terangkai sensor DHT11 dan modul wifi ESP8266 (sumber didapatkan dari powerbank) pada lingkungan pertanian atau ruangan greenhouse.
    • Hubungkan mikrokontroller dengan koneksi internet, koneksi internet ini berguna untuk mengirimkan data yang telah diperoleh dari hasil monitoring ke penyedia API thingspeak .
    • Ambil data dari API thingspeak kemudian tampilkan data suhu dan kelembaban hasil monitoring ke dalam aplikasi android yang telah dibuat.

    Sensor suhu dan kelembaban (DHT11)¶

    Sensor DHT11 adalah sensor suhu dan kelembaban udara. Range kelembaban yang dapat diukur antara 20% - 90% RH dengan tingkat akurasi ± 4% RH dan sensitivitas 1% RH. DHT11 memiliki 2 buah pin, yaitu terdiri dari pin VCC, DATA, dan GND. Berikut adalah DHT11 dapat dilihat pada gambar berikut ini:

    Modul wifi ESP8266¶

    Wifi merupakan teknologi yang digunakan pada sebuah perangkat elektronik yang mana ditujukan untuk berpindahnya data secara nirkabel atau dengan kata lain memakai gelombang radio yang melewati sebuah jaringan komputer. Peralatan dapat terhubung ke jaringan komputer melalui sebuah acces point. Sebuah acces point memiliki jangkauan kurang lebih 20 meter ketika berada dalam ruangan dan lebih luas lagi jikalau berada di luar ruangan.

    Modul Wifi ESP8266 termasuk SoC (System on Chip) dengan stack protokol TCP/IP yang mana sudah menyatu menjadi kesatuan yang utuh, sehingga tidak sulit untuk diakses dengan memakai mikrokontroler melewati komunkasi serial 802.11 b/ g/ n/ Wi-Fi Direct (P2P). Modul Wifi ESP8266 mampu menjadi host ataupun sebagai modul transfer data dalam sebuah jaringan Wifi. Modul ini memiliki kemampuan pengolahan dan penyimpanan data yang baik sehingga memungkinkan untuk diintegrasikan dengan sensor dan perangkat khusus lainnya melalui GPIO.

    Modul ESP8266 membutuhkan input tegangan dengan range 3.3 volt, namun konsumsi dayanya tinggi. Jika tegangan yang masuk kurang atau lebih dari range yang ditentukan maka modul tidak akan aktif atau kondisi yang lebih buruk lagi yaitu menjadi rusak.     Rangkaian mikrokontroller¶

    Berikut adalah desain sistem mikrokontroler meliputi perancangan arduino UNO beserta sensor DHT11 dan modul wifi ESP8266, dilihat pada gambar ini:

    Langkah untuk menyusun rangkaian sistem mikrokontroler:

    Pertama, siapkan alat berserta sensor berikut:​

    a. Mikrokontroler Arduino Uno

    b. Sensor suhu dan kelembaban (DHT11)

    c. Modul wifi ESP8266

    Kedua, hubungkan sensor DHT11 dengan mikrokontroler Arduino Uno sebagai berikut:

    a. Hubungkan kaki ke-1 DHT11 ke 5V Arduino Uno

    b. Hubungkan kaki ke-2 DHT11 ke Pin 2 Arduino Uno

    c. Hubungkan kaki ke-3 DHT11 ke GND Arduino Uno

    Ketiga, hubungkan modul wifi ESP8266 dengan mikrokontroler Arduino Uno sebagai berikut:

    a. Hubungkan kaki RX modul wifi ESP8266 ke Pin 11 Arduino Uno.

    b. Hubungkan kaki GND modul wifi ESP8266 ke GND Arduino Uno.

    c. Hubungkan kaki TX modul wifi ESP8266 ke Pin 10 Arduino Uno.

    d. Hubungkan kaki VCC dan CH_PD modul wifi ESP8266 ke 3V Arduino Uno.

    Menyiapkan akun thingspeak¶

    Pada pembuatan aplikasi diperlukan membuat akun thingspeak terlebih dahulu agar bisa mengirim dan menyimpan data hasil monitoring. Dikarenakan pada pembuatan aplikasi ini peneliti menggunakan teknologi Internet of Things (IoT) sedangkan thingspeak adalah sebuah platform Internet of Things (IoT) open source dan API yang bisa menyimpan dan mengambil data dari berbagai hal dengan memakai protokol HTTP yang melewati sebuah intenet atau melewati Local Area Network. Thingspeak dapat digunakan dalam pembuatan aplikasi pencatatan sensor dan lokasi, selain itu juga dapat digunakan dalam hal pembaruan status di jejaring sosial.

    Thingspeak memiliki dukungan terintegrasi dengan aplikasi MATLAB dari MathWork, yang memungkinkan pengguna Thingspeak untuk menganalisa dan memvisualisasikan data yang diunggah menggunakan MATLAB tanpa memerlukan pembelian lisensi MATLAB dari Mathwork.
